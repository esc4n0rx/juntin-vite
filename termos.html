1. Aceitação dos termos

2. Descrição do serviço

O Juntin é um aplicativo de gestão financeira pessoal voltado para casais. Ele permite registrar receitas, despesas e transferências, acompanhar saldos, visualizar relatórios e compartilhar um workspace financeiro com seu parceiro(a).

3. Elegibilidade

  • Você deve ter pelo menos 18 anos para criar uma conta.
  • Cada pessoa pode ter apenas uma conta ativa.
  • Cada workspace pode ter no máximo 2 membros.

4. Uso aceitável

Você concorda em utilizar o Juntin apenas para fins legais e pessoais. É proibido:

  • Usar o app para atividades fraudulentas ou ilegais.

  • Realizar engenharia reversa, descompilar ou modificar o aplicativo.
  • Compartilhar suas credenciais de acesso com terceiros.

5. Responsabilidade financeira

O Juntin é uma ferramenta de organização financeira pessoal. Não somos uma instituição financeira, não oferecemos serviços bancários, investimentos ou consultoria financeira. As informações exibidas refletem exclusivamente os dados que você e seu parceiro inserem manualmente no app.

Não nos responsabilizamos por decisões financeiras tomadas com base no uso do aplicativo.

6. Workspace compartilhado

Ao criar ou aceitar um convite para um workspace, você reconhece que:

  • Os dados financeiros inseridos ficam visíveis para todos os membros do workspace.
  • Ambos os membros podem criar, editar e excluir transações.
  • Ao sair do workspace, seus dados permanecem no histórico compartilhado.
